# Nazish Ghafoor - Portfolio Website

Personal portfolio website — Responsive design with smooth animations, custom cursor, and complete professional profile. Built with HTML, CSS, and JavaScript.

## Live Demo

Visit the live website: [https://nazishhghafoor.netlify.app/](https://nazishhghafoor.netlify.app/)

## About

A modern, fully responsive portfolio website showcasing my skills and experience as a Computer Science student, UI/UX designer, and front-end developer. The website features a clean aesthetic, smooth animations, and an intuitive user interface.

## Features

- Fully responsive design optimized for all devices (mobile, tablet, desktop)
- Custom cursor with interactive hover effects
- Smooth scroll navigation and section-based layout
- Animated skill progress bars with scroll-triggered animations
- Interactive experience timeline with hover effects
- Professional contact section with direct links to email, phone, LinkedIn, and GitHub
- Dark-themed hero section with gradient accents
- Clean, modern typography using Google Fonts

## Technologies Used

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- JavaScript (ES6)
- Intersection Observer API for scroll animations

## Sections

- Hero - Introduction and availability status
- About - Personal background, skills, and certifications
- Experience - Professional internships and roles
- Education - Academic qualifications
- Skills - Design and development expertise with percentage bars
- Contact - Direct contact information and social links
